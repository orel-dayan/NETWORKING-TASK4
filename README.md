# Networking Ex 4

 In this task , we build two programs  that use ICMP protocol.   
 a ping program , which sends a “ping” to a destination and receive a “pong”, and ping program called "safe_ping"
 that implements a timeout functionality .


## Requirements

- Ubuntu 22.04 LTS
- GNU C compiler


## Get Started

```terminal
make all
sudo ./partA <ipAdress>
sudo./partB <ipAdress>

```
